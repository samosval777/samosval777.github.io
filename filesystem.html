Файловые системы и разметка диска

NTFS (Windows)

Основные характеристики

  • Поддержка файлов до 16 ТБ
  • Журналирование
  • Разрешения доступа
  • Шифрование (EFS)

Использование

  • Основная ФС в Windows
  • Рекомендуется для системных дисков
  • Поддержка сжатия данных

Причины выбора NTFS:

  • Наиболее надежная файловая система для Windows
  • Поддержка больших файлов и разделов
  • Встроенные механизмы безопасности
  • Оптимальная производительность

Эксперимент: Сравнение скорости чтения/записи с FAT32.

ext4 (Linux)

Основные характеристики

  • Поддержка томов до 1 ЭБ
  • Отложенное выделение места
  • Журналирование

Использование

  • Стандарт для Linux-дистрибутивов
  • Поддержка прав UNIX
  • Высокая надежность

Эксперимент: Настройка квот для пользователей.

APFS (macOS)

Основные характеристики

  • Оптимизация для SSD
  • Клонирование файлов
  • Встроенное шифрование

Использование

  • Основная ФС в macOS и iOS
  • Мгновенные снимки
  • Эффективное использование пространства

Эксперимент: Создание зашифрованного контейнера.
